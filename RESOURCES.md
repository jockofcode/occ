# Useful Resources and Links

Sourced from the [kefir C project](https://git.sr.ht/~jprotopopov/kefir), the predecessor to this compiler.

## C Language Standards

- [C11 standard final working draft](http://www.open-std.org/jtc1/sc22/wg14/www/docs/n1570.pdf)
- [C17 standard final working draft](https://files.lhmouse.com/standards/ISO%20C%20N2176.pdf)
- [Clarification Request Summary for C11](https://www.open-std.org/jtc1/sc22/wg14/www/docs/n2244.htm) — defect reports incorporated into C17
- [The first C2Y draft](https://open-std.org/JTC1/SC22/WG14/www/docs/n3220.pdf) — content-wise equivalent to published C23

## System & Architecture Documentation

- [ELF Handling For Thread-Local Storage](https://www.akkadia.org/drepper/tls.pdf)
- [DWARF Version 5](https://dwarfstd.org/dwarf5std.html) — debug information format specification
- [POSIX c99 specification](https://pubs.opengroup.org/onlinepubs/9699919799/)
- [System-V AMD64 ABI](https://gitlab.com/x86-psABIs/x86-64-ABI)
- [AMD64 instruction set reference](https://www.amd.com/system/files/TechDocs/24594.pdf)
- [ARM Architecture Reference Manual (A-profile)](https://developer.arm.com/documentation/ddi0487/latest)

## Useful Tools

- [Compiler Explorer](https://godbolt.org/) — interactive compiler output explorer
- [Record and Replay framework (rr)](https://rr-project.org/) — deterministic debugging
- [CSmith](https://github.com/csmith-project/csmith) — random C program generator for compiler testing
